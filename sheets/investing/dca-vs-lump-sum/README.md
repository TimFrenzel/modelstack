# DCA Vs Lump Sum

**Category:** Investing

**Summary:** This is a clean side-by-side sheet for the classic question: put the money in now, or spread it out month by month? It uses one 24-month price path for both choices, then shows how the shares, average cost, and ending value come out. Good fit for an inheritance, bonus, or any cash pile that feels big enough to make the timing stressful.

**Structure:** `Price History` holds the monthly contribution input, the sample window, and the 24 month-end prices. `DCA Schedule` uses the same price list to track each contribution, shares bought, cumulative shares, total invested, average cost, portfolio value, and gain or loss. `Lump Sum Comparison` invests the full 24-month total up front and holds the share count constant across the same dates. `Outcome Summary` pulls the ending values, DCA average cost, lump-sum entry price, the simple notes box, and the three visuals that show value path, average cost versus price, and the contribution timing tradeoff.
