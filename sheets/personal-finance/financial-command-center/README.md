# Financial Command Center

**Category:** Personal Finance

**Summary:** This is a good sheet for someone whose money is spread across a lot of accounts and goals, and who wants one place to check the whole picture. It ties balances, cash flow, goal timing, and allocation into one workbook, which makes it easier to see if the next move should be saving more, rebalancing, or just waiting. Pretty handy when a bunch of accounts all need attention at once.

**Structure:** `Accounts` is the main registry with account type, asset class, current balance, monthly contribution, expected return, and last updated date. `Cash Flow` tracks income, fixed costs, variable costs, surplus, savings rate, and YTD surplus across 12 months. `Goals` uses `NPER` to project months and dates for six target balances. `Allocation` groups investable balances by asset class, then compares current percent against target percent and calculates the rebalance dollar amount. `Dashboard` pulls those parts together with current versus target allocation pies, a goal progress bar chart, a monthly surplus line, and a few quick KPIs at the top.
