# ModelStack

Free Excel templates for people who'd rather track their money in a spreadsheet than pay for an app. We're a small group of finance pros, Excel nerds, and hobbyist coders who got tired of seeing the same generic budget template recycled everywhere... so we started building our own.

## What you'll find

Stuff we actually use day to day (or built for friends who asked):

- Monthly budgets and expense trackers
- Net worth dashboards
- Retirement planning calculators
- Debt snowball / avalanche payoff sheets
- Portfolio trackers and rebalancing tools
- Side hustle and freelance income logs
- Options strategy P&L sheets

Some of these started as quick tools for our own workflows. Others came out of Reddit threads or Slack conversations where someone said "I wish I had a spreadsheet for that." If it's useful and deals with money or business in some way, it probably belongs here.

## Contributing

Got a spreadsheet you've been using for years? Or a Python script that generates one? We'd love to see it. Open a PR or just drop an issue with a link. Check out the [contributing guide](./CONTRIBUTING.md) if you want the details on folder structure and what to include.
